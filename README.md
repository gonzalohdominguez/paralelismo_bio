### Scripts de Imputación de Valores Faltantes

Este repositorio contiene cinco scripts en Python para realizar imputación multivariante de valores faltantes en datasets, utilizando diferentes algoritmos y enfoques de rendimiento (árboles, imputación por columnas y paralelización con multiprocessing).

### Requisitos generales

Python 3.8+
pandas
numpy
scikit-learn
openpyxl
